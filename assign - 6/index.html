About The Animations

This entire webpage is a demonstration of what can be achieved with only HTML and CSS. All the visual effects you see, from the floating title to the bouncing ball, are created using CSS @keyframes, transitions, and transforms. No JavaScript was used. This approach ensures animations are lightweight, performant, and easy to maintain.
